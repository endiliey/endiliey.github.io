Rengorum

April 30, 2019

Single-page forum application built in ReactJS and Django Rest Framework.

This is a project that I build roughly a year ago just for fun. It is open source, you can check it on https://github.com/endiliey/rengorum

#Live Demo: https://forum.endiliey.com/

#https://youtu.be/t9vWwxydREs

#Frontend

The frontend is a fast, interactive and simple Single-Page-Application (SPA), written in ES6 Javascript and built with following technologies:

  • React v16
  • Redux v4
  • React Router v4
  • Redux Thunk v2
  • Redux Persist v5.9
  • etc
